## Pass Style
Use named passes with this shape:
- Mission
- Goal
- Current state / assume but verify

- Concrete targets
- Validation
- Required output
- Done when

- Fix the smallest real trust break first.
- Separate structural correctness from felt correctness.
- Do not call a targeted probe a full certification run.
